Software

Here we keep a list of useful software:

  • edrixs – for calculating RIXS.
  • overleaf – online tool for writing papers in latex.
  • latexdiff – mark-up differences in latex files.
  • latexmk – straightforwardly compile latex in full.
  • cpdf – manipulate pdfs.
  • pdfCropMargins – crop pdfs.
  • Docker – tool for making reproducible computational environments.
  • WebPlotDigitizer – extract data from graphs.
  • diffcalc – calculate Q space i.e. a replacement for spec
